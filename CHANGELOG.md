# Changelog

## [1.0.0] - 2026-07-21

### Added
- Project foundation with Next.js 15, TypeScript, Tailwind CSS
- shadcn/ui design system with 20+ accessible components
- Authentication system (email/password) with Supabase
- Protected routes with middleware
- Zustand stores (theme, sidebar)
- TanStack Query for server state management
- Zod validation for environment config and API inputs
- Security headers and CSP configuration
- Rate limiting scaffold
- Structured logging
- Typed error handling
- Documentation (README, CONTRIBUTING, SECURITY, etc.)
- Testing infrastructure (Vitest, Testing Library)
- Cursor AI coding rules
- GitHub templates and CI configuration
- Responsive layout with sidebar navigation
- Light/dark theme support
- Versioned API structure (`/api/v1/`)
- Gemini SDK installation and configuration
